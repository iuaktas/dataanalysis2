# Funnel Analysis Project

## Overview
This project analyzes user behavior across a multi-step funnel to identify where users drop off and why.

## Business Problem
Users are completing signup but not progressing to the next step, leading to low conversion rates.

## Key Findings
- The largest drop-off occurs between signup and add-to-cart (~30–40%)
- Mobile users have lower conversion rates compared to desktop users
- Traffic source impacts user behavior, with Ads performing better than Social

## Insights
Users successfully complete the signup process but fail to experience product value afterward. This indicates a gap between onboarding and meaningful user engagement.

## Recommendations
- Improve post-signup onboarding flow with clear guidance
- Simplify the first user action to reduce friction
- Optimize mobile experience to increase conversion rates
- Focus marketing efforts on high-performing channels
